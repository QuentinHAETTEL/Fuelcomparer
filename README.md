# Fuel comparer

[![](https://img.shields.io/badge/licence-MIT-green)](https://github.com/QuentinHAETTEL/FuelComparer/blob/main/LICENSE.md)
[![](https://img.shields.io/github/v/tag/QuentinHAETTEL/fuelComparer)](https://github.com/QuentinHAETTEL/FuelComparer/blob/main/LICENSE.md)

Fuel comparer is a web application to allow you to compare fuel price in live. This is possible with OpenData

## Installation

```bash
make install
```


## Usage

### Webpack Encore compilation
```bash
make wp-watch
```

### Test code
```bash
make test
```
